3 transition

If you want add transition between normal state and hover or focus state, then add the transition property in normal state. 

input{
  border-style:solid;
  border-color:red;
  transition: border-color 1s;
}
input:focus{
    border-color:green;
}

Then there will have transition both on focus and lose focus. And for sure you can add another transition to make different effect between two states.

3.1 transition-duration

The property `transition-duration` can used not only when explicitly declared the transition, but also there are two state to transist between. Like a normal and hover state, you could just write: 

.class{
    transition-duration: 1s;
}
.class:hover{
   skdfjks: sd;

}

That will constraint the duration time when hover the element. Tips:

  • The `transition-duration` is the time from other state to this state.
  • See a detail in [[http://www.w3schools.com/css/tryit.asp?filename=trycss_buttons_animate2] [button animation]]
